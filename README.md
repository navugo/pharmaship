Pharmaship
==========

A pharmacy software for merchant ships.

This software is in active development!

## Requirements
*   Django 1.7
*   MySQL Server
*   python-mysqldb
*   python-gpgme
*   WeasyPrint (for printing PDF). Need python-lxml, python-cffi.
